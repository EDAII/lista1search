## Aplicação

### 1. Criar build
  * Por meio do terminal entre na *root* do projeto e digite o comando:
  ````  
    make
  ````

### 2. Excutar build
  * Por meio do terminal entre na *root* do projeto e digite o comando:
  ````  
    make run
  ````

### 3. Limpar build
  * Por meio do terminal entre na *root* do projeto e digite o comando:
  ````  
    make clean
  ````
